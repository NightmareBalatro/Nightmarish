## Nightmarish

>  Balatro but Nightmarishly Brütal.

Nerfs all jokers, have a hard time, you're welcome.

## Install instructions

> This mod needs `Lovely` & `Steammodded`

1. Preparations

    Backup: Copy your save data to prevent corruption.
    Locate Game Files: On Steam, right-click Balatro -> Manage -> Browse local files.
    Locate Mod Folder: Press Win+R, type `%appdata%/Balatro`, and create a `Mods` folder if it doesn't exist. 

2. Installing Core Tools (Required)

    Install Lovely: Download version.dll from the latest Lovely release and move it into your Balatro game folder (where the .exe is).
    Install Steamodded: Download the latest release of Steamodded. Extract the contents and place the smods folder into the %appdata%/Balatro/Mods folder.
    Run Game: Launch Balatro once to let the mods load. A "Mods" button should appear in the menu. 

3. Install the mod itself

    Download the [Nightmarish.zip](https://github.com/NightmareBalatro/Nightmarish/releases/) from the GitHub releases & unzip it.
    Place the `Nightmarish` folder into `%appdata%/Balatro/Mods`.
    Restart the game & the mod should show up in the main menu under mods.
